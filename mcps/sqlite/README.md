# SQLite MCP Server

This server allows an LLM to interact with a local SQLite database.

## Features
- **Query execution**: Securely run SELECT queries.
- **Schema exploration**: Discover tables and column definitions.
- **Safe access**: Restricted to a pre-defined database path.

## Setup
By default, it uses `./data/agentx.db`. You can customize this in the environment variables after installation.
